# AI Learning

这是一个用于系统学习人工智能相关知识的个人仓库，主要用于沉淀学习笔记、代码示例、实验 demo 以及阶段性总结。项目会从基础概念和常用框架开始，逐步扩展到大模型、Agent、RAG、AI 应用开发以及前沿方向探索。

## 项目简介

本仓库当前主要包含 LangChain 和 LangGraph 的基础学习内容，包括框架入门、核心概念、基础用法和示例代码。后续会持续补充大模型基础、提示词工程、Agent 设计、工具调用、检索增强生成、向量数据库、模型评估以及 AI 应用 demo 等内容。

这个项目的目标是将 AI 学习过程结构化、可复盘、可持续更新，方便后续查阅、总结和实践。

## 目录结构

```text
AILearning/
├── langchain/      # LangChain 基础知识、核心概念和示例
├── langgraph/      # LangGraph 基础知识、工作流和 Agent 图示例
├── test.py         # 临时测试脚本
└── README.md       # 项目说明文档
```

## 学习范围

计划持续学习和整理的方向包括：

- LangChain 基础与应用模式
- LangGraph 工作流与 Agent 状态管理
- 大语言模型基础概念与 API 使用
- Prompt Engineering 与结构化输出
- AI Agent 设计与工具调用
- RAG 检索增强生成
- Embedding 与向量数据库
- 模型评估、监控与可观测性
- AI 应用开发 demo 与实验项目
- AI 前沿方向、论文与技术趋势整理

## 使用说明

本仓库主要用于个人学习和实践，内容会随着学习进度持续更新。仓库中的示例代码和笔记可能会不断调整，不保证所有内容都处于最终稳定状态。

常见内容形式包括：

- Python 脚本：用于保存可运行的示例代码
- Jupyter Notebook：用于交互式学习、实验和记录
- Markdown 文档：用于整理概念、笔记和阶段总结
